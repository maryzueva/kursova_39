Назва проєкту: Task39 — Обробка числових сіток з обмеженнями

Опис:
Програма розв'язує задачі на основі сітки:
1. Сітка з цифрами (Puzzle): кожне число означає розмір білої області
2. Сітка без цифр (Fixed): потрібно уникати чорних блоків 2x2

Меню:
- Вибір типу задачі
- Варіант вводу: вручну або з input.txt
- Графічне виведення з підсвіткою

Файли:
- task39.cpp — головне меню
- puzzle.cpp / puzzle.h — клас Puzzle
- fixed_board.cpp — логіка варіанту без цифр
- README.txt — опис

Компіляція:
g++ -Wall task39.cpp puzzle.cpp fixed_board.cpp -o task39

Запуск:
./task39
